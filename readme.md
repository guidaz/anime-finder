# 💻API pra Buscar Animes

Projeto desenvolvido com **HTML, CSS e JavaScript** com o objetivo de consumir dados externos de uma API pública de animes (Jikan API), manipular o estado local de um navegador (localStorage) e apresentar uma interface organizada e documentada.

> Curso Análise e Desenvolvimento de Sistemas - Fatec Itu

## 🚀Tecnologias Utilizadas 
    - HTML5
    - CSS3
    - JavaScript Moderno (ES6+)
    - SweetAlert2

## ⚡Funcionalidades do Projeto
    - Buscar animes por nome.
    - Visualizar detalhes de cada anime, como os principais títulos, sua Sinopse traduzida e Avaliação Geral dos usuários.
    - Adicionar e remover favoritos, armazenando-os localmente no localStorage.
    - Interface limpa e organizada, responsiva e fácil de usar.

## 🧠Melhorias de Experiência do Usuário (UX)
    - Exibição de alerta quando o campo de busca está vazio.
    - Feedback visual ao adicionar um anime aos favoritos.
    - Substituição de alert() padrão por SweetAlert2 para mensagens mais amigáveis.
    - Confirmação antes de remover um anime da lista de favoritos.
